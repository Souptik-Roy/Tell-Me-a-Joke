# Tell-Me-a-Joke[https://tellme-a-joke.netlify.app/]

## Description
'Tell Me a Joke' is a simple web app that displays random Indian-themed jokes on button clicks using JavaScript.

## Features
- Generates a random joke from a predefined list.
- Jokes change with each button click.
- Simple and interactive UI.

## Files
- **tellMeJokes.html**: Webpage structure.
- **tellMeJokes.css**: Styling.
- **sample.js**: Manages joke selection and button interactions.

## Usage
1. Open `tellMeJokes.html` in a browser.
2. Click the 'Click Me' button.
3. A random joke appears.
   
## Methods:
Method 1: Displays a joke via an alert box.<br>
Method 2: (Planned) Updates the joke text directly on the webpage.

## Future Improvements
- Prevent duplicate jokes.
- Implement Method 2 to display jokes without alerts.
- Enhance UI with animations.
